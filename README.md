 Repositório criado com o objetivo de armazenar e compartilhar os códigos do nosso projeto do sistema de incêndio para o IFRN campus Lajes.

Esse repositório é privado, apenas para os integrantes do projeto, e será usado para maior flexibilidade no compartilhamento dos scrripts usados para o funcionamento do servidor e dos ESP32

## Objetivos:
- Armazenar os arquivos dos scripts usados no projeto do TCC Fogo e os arquivos .sql para salvar a estrutura do banco de dados utilizado;
- Acompanhar as mudanças do projeto pelos commits;
- Facilitar o compartilhamento dos scripts do projeto;
- Facilitar o acesso aos arquivos do projeto;
